固件协议说明

ESP32 需烧录固件，通过串口以 JSON 格式每秒发送数据：

{
  "humidity": 65.2,
  "temperature": 24.8,
  "timestamp": 1717123456
}

或通过 WebSocket 在端口 81 上广播相同格式的 JSON 数据。
